# Welcome to DriftWM

**DriftWM** is a modern, spatial, infinite-canvas Wayland window manager. This site documents my personal, highly-optimized DriftWM configuration.

## 🚀 Core Features of This Setup
- **Infinite Canvas Navigation**: 11 spatial canvas bookmarks mapped to `Mod+1` through `Mod+=` for instant workspace-like jumping.
- **Custom Rofi Integration**: Launchers and power menus styled with the `fancy.rasi` theme.
- **Waybar Integration**: Custom `config.jsonc` and `style.css` running via autostart.
- **Magnetic Snapping**: Window clustering and edge snapping enabled for fluid tiling-like behavior on a spatial canvas.

## 📂 Configuration Structure
All configuration is managed via `~/.config/driftwm/config.toml`. 
Key additions in this setup:
- **Background**: Custom wallpaper (`Bluespaceship.jpg`).
- **Autostart**: Waybar, `xdg-desktop-portal`, and `nm-applet`.
- **Screenshots**: Bound to `Print` key, saving directly to `~/Pictures/Screenshots/driftwm/`.
